5. VM LIFECYCLE BEHAVIOR UNDERSTANDING

Tested Vagrant commands:

vagrant halt

* Shuts down VMs
* Does NOT reinstall
* Preserves state

vagrant up

* Restarts existing machines
* Does NOT re-run provisioning

vagrant destroy

* Deletes VMs completely
* Removes all state
* Next vagrant up rebuilds from scratch

Conclusion:
For reproducibility and clean testing, use:

```
vagrant destroy -f
vagrant up
```
